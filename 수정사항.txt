--------------------{해결됨}-------------------

유저분들이 헛간 GUI 에서 농작물을 저장할때 몇씩 포인트가 쌓이는지 알수가없음
아이템 설명칸에 몇포인트가 쌓이는지 볼수있는 기능이 필요

--------------------{해결됨}-------------------
헛간 초기화 12시간제 도입
config.yml 에
```
daily_reset_time: "00:00" #  00시 초기화, 12시 초기화
```
은 첫 리셋 시간으로 변경
```
daily_reset_hour: 12 # 12시간마다 초기화 진행
``` 12시간마다 초기화 진행

만약 다음과 같이 설정 되어있다면
```
daily_reset_time: "5:00"
daily_reset_hour: 6
``` 5시, 10시, 16시, 22시 마다 초기화 진행

헛간 할당량은 있되 할당량 기준치를 초과해서 등록 가능하게 수정 (초기화 진행은 동일, 초과는 다음때 반영안됨)

--------------------{문제점}-------------------
현제 작동은 잘 되나 문제점이 있음 초기화 시간을 23시 25분으로 설정하고 23:24에 농작물을 1500포인트 저장 했는데 무승부 출력됨
